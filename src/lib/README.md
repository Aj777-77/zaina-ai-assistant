# Library / Utilities

This directory contains utility functions and configurations for external services.

## Contents

- **Firebase**: Firestore database configuration and helper functions
- **OpenAI**: OpenAI API integration and utilities
- **Helpers**: Shared utility functions
- **Constants**: Application-wide constants

## Usage

Import utilities as needed:
```typescript
import { db } from '@/lib/firebase';
import { openai } from '@/lib/openai';
```
